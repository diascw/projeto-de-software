Principais atividades realizadas na fase de projeto 
- Detalhamento dos aspectos dinâmicos do sistema
- O refinamento dos aspectos dinâmicos e estruturais do sistema
- Detalhamento da arquitetura do sistema
- Definição das estratégias para armazenamento, gerenciamento e persistência dos dados manipulados pelo sistema.

+ Realização do projeto da interface gráfica com o usuário
+ Definição dos algoritmos a serem utilizados na implementação


O projeto de software é composto d eduas atividades
* Projeto de arquitetura de software
- Projeto mais alto nível com granularidade macro do software definindo seus componentes e a interface de comunicação
- objetivo é satisfazer os requisitos de qualidade 

* O projeto detalhado do software
- projeto mais baixo nível com granularidade micro do software definindo seus objetos e a forma de colaboração entre eles para realizar as funções do sistema
- objetivo é satisfazer os requisitos funcionais

* ARQUITETURA 
- componentes e conectores
- restrições sobre componentes e conectores
- composição de componentes

* PROJETO (detalhamento)
- procedimento e interfaces
- algoritmos e estruturas de dados
- composição procentual
